# I like this world (without Ad)

シンプルな広告ブロッカー Chrome拡張機能です。
主要な広告・トラッキングドメインをブロックし、悪質なポップアップ広告(マウス追従・全画面クリックトラップなど)も無効化します。

## 機能

- **確定リストによる広告ブロック**(28,000以上のドメイン)
- **拡張モード / 拡張+モード**: ドメイン名の特徴をスコアリングして、未知の広告ドメインも推定ブロック
- **サイトごとのON/OFF**: 特定のサイトだけ広告ブロックを無効化できます
- **悪質なポップアップ・クリックトラップ対策**: カーソル追従広告や全画面の透明クリック誘導要素を無効化
- ブロック数や検知状況をツールバーのポップアップで確認できます

## インストール方法

このリポジトリは Chrome Web Store には公開していないため、「デベロッパーモード」を使って手動でインストールします。

### 1. ダウンロード

- このリポジトリの **Releases** から最新の `my-adblock.zip` をダウンロードする
- または、緑色の `Code` ボタン → `Download ZIP` でリポジトリ全体をダウンロードする

### 2. 展開

ダウンロードした zip を、**好きな場所(削除しないフォルダ)** に展開してください。
例: `C:\extensions\my-adblock` や `~/extensions/my-adblock`

> ⚠️ 展開したフォルダを後から移動・削除すると、拡張機能が無効になります。

### 3. Chromeに読み込む

1. Chromeで `chrome://extensions` を開く
2. 右上の「デベロッパーモード」をONにする
3. 「パッケージ化されていない拡張機能を読み込む」をクリック
4. 展開した `my-adblock` フォルダ(`manifest.json` が直下にあるフォルダ)を選択する

これでツールバーにアイコンが表示され、有効化されます。

## 使い方

ツールバーのアイコンをクリックするとポップアップが開きます。

| 項目 | 説明 |
|---|---|
| このサイトの広告をブロックする | 今見ているサイトだけ広告ブロックをON/OFFできます |
| 基本モード | 確定リストのドメインのみブロック(推奨・安全) |
| 拡張モード | 確定リスト＋広告と強く推定されるドメインもブロック(慎重判定) |
| 拡張+モード | 判定を緩めて、より広く推定ブロック(積極判定・誤爆の可能性あり) |
| オフ | 広告ブロックを全て無効化 |
| ブロック済みリクエスト | このページでブロックした通信の数 |
| 怪しい通信(未ブロック) | 広告の可能性があるが確定ではないドメインの一覧 |

モードやサイトごとの設定は保存され、次回も保持されます。

## 更新方法

1. 新しい `my-adblock.zip` をダウンロードして展開し、既存のフォルダに上書きする
2. `chrome://extensions` を開き、本拡張機能の更新ボタン(🔄)をクリックする

## 不具合・要望

特定のサイトで表示が崩れる、広告が消えない等があれば、このリポジトリの Issue で報告してください。
ブロックして欲しい広告ドメインが分かれば、ブラウザの開発者ツール(Network タブ)で確認したドメイン名を教えてもらえると追加しやすいです。

## 注意事項

- 本拡張機能は個人利用・少人数配布を目的としています
- サイトによっては、広告ブロックを検知して警告を表示することがあります。その場合は「このサイトの広告をブロックする」をOFFにするか、サイト側の指示に従ってください
- 「拡張+モード」は誤検知でサイトの一部機能(画像・スクリプト等)が正しく表示されなくなる場合があります。問題が出た場合は基本モードまたは拡張モードに切り替えてください
