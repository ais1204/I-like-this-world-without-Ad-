# I like this world (without Ad)

シンプルな広告ブロッカー Chrome拡張機能です。
主要な広告・トラッキングドメインをブロックし、悪質なポップアップ広告(マウス追従・全画面クリックトラップなど)も無効化します。

## オープンソースについて

このリポジトリのコードは全て公開されています。

- `manifest.json` / `background.js` / `content.js` / `popup.js` など、動作に関わる全ファイルをこのリポジトリで確認できます
- ユーザーの閲覧履歴やデータをどこかに送信するような処理は含まれていません(ネットワーク通信を行うコードがないことは、`background.js`・`content.js`を見れば確認できます)
- ブロック対象のドメインリスト(`rules.json` / `blocklist.js`)もすべて中身が見られるテキストファイルです

ブラウザの拡張機能は強い権限(閲覧中のページの内容やすべての通信)を持つため、特に広告ブロッカーのように「全サイトを見る」拡張機能は、提供元が信頼できるか・コードが公開されているかを確認することをおすすめします。このプロジェクトは個人が少人数向けに公開しているものなので、第三者による正式なセキュリティ監査は受けていませんが、気になる方はコードを直接読んで判断してください。

本拡張機能のソースコードは MIT License で公開しています(詳細は [LICENSE](LICENSE) ファイル)。なお、同梱のドメインリスト(`rules.json` / `blocklist.js`)は事実としてのドメインデータを自動・AI支援で収集したもので、公開ブロックリストと重複する可能性があり、無保証で提供されます。

## リポジトリの構成

リポジトリ直下に拡張機能のソースファイル一式(`manifest.json`、`background.js`など)を置いています。`git clone`してそのまま `chrome://extensions` から読み込むこともできますが、**安定版が欲しい場合は上記の通り Releases の zip を使ってください**。直下のファイルは開発中で、動作が不安定な場合があります。

## 機能

- **確定リストによる広告ブロック**(28,000以上のドメイン)
- **拡張モード / 拡張+モード**: ドメイン名の特徴をスコアリングして、未知の広告ドメインも推定ブロック
- **サイトごとのON/OFF**: 特定のサイトだけ広告ブロックを無効化できます
- **悪質なポップアップ・クリックトラップ対策**: カーソル追従広告や全画面の透明クリック誘導要素を無効化
- ブロック数や検知状況をツールバーのポップアップで確認できます

## インストール方法

このリポジトリは Chrome Web Store には公開していないため、「デベロッパーモード」を使って手動でインストールします。

### 1. ダウンロード

**Releases からのダウンロードを推奨します。**[Releases](../../releases) ページから最新版の `my-adblock.zip` をダウンロードしてください。`main`ブランチ(リポジトリ直下のファイル)は開発中で変更される場合があるため、安定して使いたい場合は必ず Releases の最新版を使ってください。

> Releases を使わない場合は、緑色の `Code` ボタン → `Download ZIP` でリポジトリ全体をダウンロードすることもできますが、これは開発中の最新コードです。

### 2. 展開

ダウンロードした zip を、**好きな場所(削除しないフォルダ)** に展開してください。
例: `C:\extensions\my-adblock` や `~/extensions/my-adblock`

> ⚠️ 展開したフォルダを後から移動・削除すると、拡張機能が無効になります。

### 3. Chromeに読み込む

1. Chromeで `chrome://extensions` を開く
2. 右上の「デベロッパーモード」をONにする
3. 「パッケージ化されていない拡張機能を読み込む」をクリック
4. 展開してできたフォルダの中から、`manifest.json` が直下にあるフォルダを選択する
   - Releases の zip の場合: 展開すると `my-adblock` フォルダの中に `manifest.json` があります
   - リポジトリ全体の zip の場合: 展開してできたフォルダ自体の直下に `manifest.json` があります

これでツールバーにアイコンが表示され、有効化されます。

## 使い方

ツールバーのアイコンをクリックするとポップアップが開きます。

| 項目 | 説明 |
|---|---|
| このサイトの広告をブロックする | 今見ているサイトだけ広告ブロックをON/OFFできます |
| 基本モード | 確定リストのドメインのみブロック(推奨・安全) |
| 拡張モード | 確定リスト＋広告と強く推定されるドメインもブロック(慎重判定) |
| 拡張+モード | 判定を緩めて、より広く推定ブロック(積極判定・誤爆の可能性あり) |
| オフ | 広告ブロックを全て無効化 |
| ブロック済みリクエスト | このページでブロックした通信の数 |
| 怪しい通信(未ブロック) | 広告の可能性があるが確定ではないドメインの一覧 |

モードやサイトごとの設定は保存され、次回も保持されます。

## 更新方法

1. [Releases](../../releases) から最新の `my-adblock.zip` をダウンロードして展開する
2. 展開した `my-adblock` フォルダの内容を、既存の拡張機能フォルダにそのまま上書きする
3. `chrome://extensions` を開き、本拡張機能の更新ボタン(🔄)をクリックする

## 不具合・要望

特定のサイトで表示が崩れる、広告が消えない等があれば、このリポジトリの Issue で報告してください。
ブロックして欲しい広告ドメインが分かれば、ブラウザの開発者ツール(Network タブ)で確認したドメイン名を教えてもらえると追加しやすいです。

## 注意事項

- 本拡張機能は個人利用・少人数配布を目的としています
- サイトによっては、広告ブロックを検知して警告を表示することがあります。その場合は「このサイトの広告をブロックする」をOFFにするか、サイト側の指示に従ってください
- 「拡張+モード」は誤検知でサイトの一部機能(画像・スクリプト等)が正しく表示されなくなる場合があります。問題が出た場合は基本モードまたは拡張モードに切り替えてください
